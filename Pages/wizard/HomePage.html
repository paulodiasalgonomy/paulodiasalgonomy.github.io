Recs For Placements

Description
This endpoint returns recommendations for a named placement and logs shopper behavior. The recommendations returned by the API respect all merchandising rules set up in the Omnichannel Personalization dashboard.

Endpoint
https://<environment>.richrelevance.com/rrserver/api/rrPlatform/recsForPlacements
Returns content for the given user and placement.

Required home page parameters

placements


Data Type: String | List of placement identifiers separated by ‘|’. Each identifier consists of a page type (must begin with home_page) and a placement name. You will get one set of recommendations for each placement. The first placement is assumed to be the “best” placement and will receive the best recommendation strategy. When multiple placements are requested in the same call, each will receive a unique strategy and unique products.

Capturing clickstream behavior
Only specify home_page to report home page views without returning recommendations.
